# Projeto e-commerce

Este é um projeto em andamento, com propósito de aprendizado, desenvolvido através de curso da Udemy. 
O projeto receberá commits pequenos e frequentes para desenvolver a prática de comandos Git.

### Créditos
Projeto desenvolvido do zero no [Curso de PHP 7](https://www.udemy.com/curso-completo-de-php-7/) disponí­vel na plataforma da Udemy e no site do [HTML5dev.com.br](https://www.html5dev.com.br/curso/curso-completo-de-php-7).

Template usado no projeto [Almsaeed Studio](https://almsaeedstudio.com)

### Tecnologias e ferramentas utilizadas
* MySql (com Workbench) - banco de dados da aplicação
* PHP - para o back-end
* Wampserver - para configuração e gerenciamento do PHP + Apache
* Composer - para gerenciamento de dependências
* Raintpl - para construção templates em PHP
* Slim framework - microframework para aplicações web em PHP

### Bibliotecas e templates utilizados
* PHPmailer - biblioteca PHP para envio de e-mails
* AdminLTE - template para a área de admin
